# Insta Store ERD

This repository contains the Entity Relationship Diagram (ERD) and schema structure for the Insta Store project.

## Included Entities / Schemas
- products
- customers
- orders
- payments
- shipments
- thrifted_products
- handmade_products
- order_items

## Purpose
To document and visualize how core entities connect in the application.

## Cardinality Legend
In cardinality, green line represents 1:1 and pink represents 1:N.
